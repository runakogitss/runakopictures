# Runako Pictures - Vintage Photography Gallery

A beautiful vintage-themed photography portfolio website showcasing fine art photography collections.

## 🎨 Features

- **Responsive Gallery Layout** - Justified gallery layout that adapts to all screen sizes
- **Lightbox Viewer** - Full-screen image viewing with navigation
- **Vintage Aesthetic** - Classic design with elegant typography and ornamental elements
- **Multiple Collections** - Organized galleries for different photography themes
- **Image Optimization** - Automatic image resizing and format conversion
- **Lazy Loading** - Images load as needed for better performance
- **Mobile Optimization** - Smaller images and better touch support
- **Loading Animations** - Skeleton screens while images load
- **Batch Loading** - Images load in batches for better UX
- **Responsive Images** - Different sizes for different screen sizes

## 📸 Gallery Collections

- **Japan Trip** - Intimate exploration of Japan's ephemeral beauty
- **Kota Lama** - Historic colonial architecture of Semarang's old town
- **School Life** - Capturing precious memories of high school experiences
- **Studiocology** - Social project documenting educational initiatives

## 🛠️ Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Custom styling with vintage design elements
- **JavaScript** - Interactive gallery functionality
- **Flickr Justified Gallery** - Responsive gallery layout
- **Advanced Gallery System** - Custom image optimization and lazy loading
- **Image Utils** - Automatic image resizing and format conversion
- **Intersection Observer API** - Efficient lazy loading implementation
- **Google Fonts** - Playfair Display, Crimson Text, Old Standard TT

## 🚀 Getting Started

### Prerequisites
- Node.js (for development server)
- Modern web browser

### Installation

1. **Install Git LFS** (required for images):
```bash
# Install Git LFS first
git lfs install
```

2. Clone the repository:
```bash
git clone https://github.com/yourusername/runako-pictures.git
cd runako-pictures
```

3. **Download LFS files** (photography images):
```bash
git lfs pull
```

4. Install dependencies:
```bash
npm install
```

5. Start development server:
```bash
npm run dev
```

6. Open your browser and navigate to `http://localhost:3000`

### 📸 Git LFS for Images

This project uses **Git LFS** to efficiently manage photography files:
- **80+ high-resolution images** stored in LFS
- **Faster repository cloning** without downloading all images
- **Optimized bandwidth usage** for contributors
- See `GIT_LFS_SETUP.md` for detailed setup instructions
